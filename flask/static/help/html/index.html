SELECT A SENSOR

   You can select and view data from one sensor at a time.

   Sensor icons are placed on the map in locations where spectrum data is available. Properties of a sensor will pop up when you place the mouse cursor over a sensor icon.

   Select 'Show Markers by Freq Band' or 'Show Markers by Detected System' from the map page main menu bar to narrow down Sensor selection.

   Left click on a sensor icon to select that sensor. The selected sensor will change color from red to yellow.

   Information about the sensor you click on will appear to the left of the map:
     you must click on one of the available frequency bands listed at the bottom to continue on to view data.

VIEW DATA: SELECTION OF START DATE AND DURATION

   After selecting a sensor and frequency band to view, you will see a map with sensor information listed to the left.

   Start date, number of days and number of aquisitions for the band you selected is shown above the map.

   From the menu bar above the map, left click on 'Generate Daily Occupancy Chart' to continue and view plots, or 'Download Data' to retrieve a data file.
